# Intersection of Two Arrays II

最佳代码
```cpp
unordered_map<int, int> map;
for (int& num : nums1) 
    map[num]++; //插入unordered_map元素时可以用这种方法，不需检查该值是否存在，如果不存在会自动插入

vector<int> ret;
for (int& num : nums2) {
    if (map.find(num) != map.end()) {
        ret.push_back(num);
        map[num]--;
        if (map[num] == 0) 
            map.erase(num);
    }
}
return ret;
```
