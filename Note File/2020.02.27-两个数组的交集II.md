# Intersection of Two Arrays II

最佳代码
```cpp
unordered_map<int, int> map;
for (int& num : nums1) map[num]++;

vector<int> ret;
for (int& num : nums2) {
    if (map.find(num) != map.end()) {
        ret.push_back(num);
        map[num]--;
        if (map[num] == 0) map.erase(num);
    }
}
return ret;
```
