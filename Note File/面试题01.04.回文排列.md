# Palindrome Permutation LCCI

知识点：  
bitset、哈希表

## 最佳代码
```cpp
class Solution {
public:
    bool canPermutePalindrome(string s) {
        bitset<128> b_set; //设置bitset为128是因为ASCII码表共有128个字符
        for (char c : s) {
            b_set.flip(c);
        }
        return b_set.none() || b_set.count() == 1;
    }
};
```
